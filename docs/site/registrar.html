Accu-DID Registrar API (0.9.0)

Download OpenAPI specification:

URL: https://github.com/opendlt/accu-did License: MIT

DID registrar for did:acc on Accumulate devnet. Supports both native endpoints and DIF Universal Registrar-compatible endpoints for complete DID lifecycle management. Operation Modes: - FAKE: Uses mock submitter for offline testing - REAL: Connects to Accumulate blockchain via JSON-RPC v3 API Patterns: - Native endpoints: Clean, direct API for Accumulate DID operations - Universal endpoints: DIF-compatible for interoperability with Universal Registrar infrastructure

Create a new did:acc DID

Creates a new DID with complete Accumulate transaction flow: 1. Create ADI (if needed) 2. Create data account 3. Write initial DID Document

Request Body schema: application/json
required
required
object (DIDDocument)

A W3C DID Core compliant DID Document

object

Creation options

Responses

Local development server

http://127.0.0.1:8081/register

Local development server (alternate)

http://localhost:8081/register

Deactivate a DID

Deactivates a DID by writing a tombstone entry to the data account. Once deactivated, the DID cannot be reactivated.

required
did
required
string^did:acc:.+

required
boolean
Value: true
